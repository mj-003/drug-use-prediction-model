About the project

This project analyzes relationships between drug consumption patterns, personality traits, demographic characteristics, and mental health levels.

Using advanced machine learning techniques, the predictive model identifies risk factors associated with psychoactive substance use and provides valuable information for addiction research.

The project combines traditional data analysis techniques with modern generative artificial intelligence (GenAI) methods, creating a comprehensive solution.
